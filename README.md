# eyedle — your idle screens, handled

**[eyedle.io](https://eyedle.io)**

Eyedle watches where you look. The monitor you're looking at runs at full
power — the rest dim and throttle, and snap back the instant you glance over.
Like peripheral vision, for your desk.

- **[Download for Windows](https://github.com/aalexander1088-svg/eyedle/releases/latest/download/eyedle-win64.zip)** — free · Windows 10/11 · 2+ monitors · webcam optional
- Everything runs on-device. Nothing recorded, nothing uploaded, no telemetry.
- Free for personal use. Business/team deployment: eyedleteams@gmail.com

This repository hosts the website and release downloads.
